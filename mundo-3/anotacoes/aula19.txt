# Título da aula: Dicionários

Os dicionários são variáveis compostas como as tuplas e listas, mas que permitem índices literais.
Os nomes dados para os elementos são "Keys" (chaves)

Definição de dicionários:
    dados = {}
    dados = dict()

Adicionar elementos no dicionários
    - Não há o método append. Basta criar o elemento e atribuir valor.
    Ex:
        dados = dict()                       => Definição do dicionário vazio.
        dados = {'nome':'Pedro', 'idade':25} => Atribuição de valores ao dicionário
        dados['sexo'] = 'M'                  => Adiciona o novo elemento 'sexo' com valor 'M'.

Remover elementos
    del dados['idade']

Exibição do conteúdo
    É possível exibir três tipos de contéudos: Vaalores, Chaves e os Itens.
    Ex:
        print(filme) -> Exibe o dicionário inteiro
        >>> {'nome': 'Star Wars', 'ano': 1977, 'diretor': 'George Lucas'}

        print(filme.values()) -> Exibe apenas os valores
        >>> dict_values(['Star Wars', 1977, 'George Lucas'])

        print(filme.keys()) -> Exibe apenas as chaves
        >>> dict_keys(['nome', 'ano', 'diretor'])

        print(filme.items()) -> Exibe os itens relacionados (3 tuplas)
        >>> dict_items([('nome', 'Star Wars'), ('ano', 1977), ('diretor', 'George Lucas')])
