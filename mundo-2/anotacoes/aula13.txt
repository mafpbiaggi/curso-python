# Título da aula: Estrutura de Repetição

Laços == Repetições == Iterações == Loopings


Exemplo: 
    laço i no intervalo (1,10)
        Comando

    for c in range(1,10):
        print(c)

Obs: O intervalo do laço vai até i-1, ou seja, num range(1,10), o último i será 9.

Prática

    Contagem regressiva
        for c in range(6, 0, -1)

    Saltando entre números
        for c in range(0, 7, 2) # faz a contagem de 2 em 2

    Chegando em número 'n'
        n = int(input('Digite um número: '))
        for i in range(0, n+1):
            print(c)

    Coletando os parâmetros
        i = int(input('Início: '))
        f = int(input('Fim: '))
        p = int(input('Passo: '))
        for c in range(i, f+1, p):
            print(c)
        print('Acabou')

    Somatório
        s = 0
        for c in range(0, 4):
            n = int(input('Digite um valor: '))
            s += n # Equivale à s = s + n
        print('O somatório de todos os valores foi {}.'.format(s))
