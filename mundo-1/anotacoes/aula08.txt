# Título da aula: Utilizando módulos

Um módulo é um programa separado do código principal (em apenas um arquivo) que tem funções específicas.

Por exemplo:
- Módulo bebida (bebida.py)
-- Contém as funções: agua, refrigerante, suco, chá, leite.

    # arquivo bebida.py
    def agua(c):
        if c == 1:
          return 'potável'

    def refrigerante(b)
        if b == 2
            return 'coca-cola'

Importação de módulos

1. Completo: import bebida
2. Função específica: from bebida import agua
3. Funções específicas: from math import sqrt, ceil

Usando a opção 1, o programa em Python carregará todas funções do módulo bebida; Já na opção 2, apenas a função agua será adicionada.


Vantagem de se importar apenas uma função
- Menos utilização de memória.

Biblioteca é um conjunto de módulos, portanto, um pacote com vários arquivos:

bebidas/
├── agua/
├── refrigerante/
├── cha/
└── ...

Exemplo:
# Realizar a raíz quadrada de um número usando a biblioteca math

    import math
    num = int(input('Digite um número: '))
    raiz = math.sqrt(num)
    print('A raíz de {} é igual a {:.2f}'.format(num, raiz))

ou

    from math import sqrt
    num = int(input('Digite um número: '))
    raiz = sqrt(num) # Não é necessário chamar o método math, basta chamar a função.
    print('A raíz de {} é igual a {:.2f}'.format(num, raiz))
